# Computica's Faction Forks

**Restoring abandoned factions • Translating Chinese mods • Cleaning code • Bridging compatibility • Polishing gameplay**

Computica's Faction Forks is a Starsector mod rehabilitation project focused on restoring older faction mods, translating Chinese-language mods into proper English, optimizing code, and polishing rough or unfinished content.

The goal is not to erase the identity of the original mods. The goal is to preserve good ideas, make them easier to maintain, and help them survive in modern Starsector modlists.

## Project Focus

- Restore older faction mods that are outdated, broken, abandoned, or difficult to run.
- Translate Chinese mods into proper English with natural phrasing.
- Clean and optimize code where practical.
- Improve CSV, JSON, faction, ship, weapon, hullmod, and variant data.
- Apply balance and polish tweaks when something is clearly broken, awkward, or unfinished.
- Bridge compatibility with major non-faction mods.

## Compatibility Bridging

This project also aims to improve compatibility with major campaign-layer and ecosystem mods, including:

- Ashes of The Domain
- Industrial.Evolution
- Nexerelin
- Star Lords

Compatibility support may include market adjustments, economy checks, industry integration, diplomacy review, invasion behavior review, and corrections for outdated or broken mod interactions.

Compatibility support does not guarantee every possible mod combination. The goal is practical stability and cleaner behavior in common Starsector mod setups.

## Development Tools

This is a human-led project with Codex-assisted development.

Codex may be used for code review, refactoring assistance, bug hunting, JSON/CSV cleanup, scripting support, and identifying possible optimization paths. Final decisions, testing priorities, balance judgment, and release direction are handled manually.
